◆ opsick_client_return_code

enum opsick_client_return_code

An enumeration containing all possible return values that the opsick client library may use (starting from major version 2 upwards).

Enumerator
OPSICK_CLIENT_SUCCESS 

Return code of a successful opsick client function execution.

OPSICK_CLIENT_FAILURE 

Return code for generic failures within the library.

OPSICK_CLIENT_INVALID_ARGS 

This is returned when any invalid argument(s) were passed into an opsick client function.

OPSICK_CLIENT_OUT_OF_MEMORY 

Uh oh...

OPSICK_CLIENT_UNINITIALIZED 

This return code is returned if a function call was made without having called opsick_client_init() at least once.

OPSICK_CLIENT_INIT_FAILED 

This error code describes library initialization failures.

See also
opsick_client_init()
OPSICK_CLIENT_INVALID_SERVER_URL 

Returned when the used server URL is invalid.

OPSICK_CLIENT_SERVER_URL_TOO_LONG 

Returned when the used server URL is too long.

OPSICK_CLIENT_CONNECTION_TO_SERVER_FAILED 

This error code describes a failure to connect to the established opsick URL.

OPSICK_CLIENT_CONNECTION_TO_SERVER_WEIRD 

Returned if things got weird.... (connection to the server OK but server doesn't seem to be an Opsick server).

OPSICK_CLIENT_REQUEST_SUBMISSION_TO_SERVER_FAILED 

Error code for when submission of the HTTP request to the Opsick server failed.

OPSICK_CLIENT_INVALID_SERVER_RESPONSE_FORMAT 

Returned if the server's response was not in a valid format.

OPSICK_CLIENT_INVALID_SERVER_RESPONSE_SIGNATURE 

Returned if the server's signature couldn't be verified against its response (uh oh...)

OPSICK_CLIENT_KEY_REGEN_NOT_PERFORMED 

Manual key regeneration not performed due to either failure or not being necessary yet.

OPSICK_CLIENT_MISSING_PRIVATE_KEYS 

Operation failed due to the used opsick_client_user_context not containing private keys.

OPSICK_CLIENT_PWCRYPT_ENCRYPTION_FAILED 

This error code is returned when symmetric encryption using pwcrypt failed.

OPSICK_CLIENT_PWCRYPT_DECRYPTION_FAILED 

Symmetric decryption using pwcrypt failed (e.g. wrong password).

OPSICK_CLIENT_CECIES_ENCRYPTION_FAILED 

Error code for when the asymmetric encryption of the message for the server failed.

OPSICK_CLIENT_CECIES_DECRYPTION_FAILED 

This is returned if asymmetric decryption of the server's response failed.

◆ opsick_client_test_connection()

OPSICK_CLIENT_API int opsick_client_test_connection ( const char *  server_url)

Tests the connection to an opsick server.

Parameters
server_urlThe opsick server base URL.
Returns
  • OPSICK_CLIENT_SUCCESS on success
  • OPSICK_CLIENT_INVALID_SERVER_URL in case of an invalid server_url (these require explicit http:// or https:// protocol prefix and NO trailing slashes!)
  • OPSICK_CLIENT_SERVER_URL_TOO_LONG in case the server_url exceeds the maximum allowed number of characters.
  • OPSICK_CLIENT_CONNECTION_TO_SERVER_FAILED if connection couldn't be established successfully.
  • OPSICK_CLIENT_CONNECTION_TO_SERVER_WEIRD if the connection to the URL was OK but the server is... weird... it kinda doesn't seem to be an opsick server...
  • (if none of the above opsick_client_return_code return codes are returned, the HTTP response status code is returned)
